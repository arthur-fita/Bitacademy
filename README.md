# 📚 BitAcademy — Portal de Aprendizado

## 🧠 Sobre o Projeto

O **BitAcademy** é um portal de aprendizado desenvolvido como projeto acadêmico por uma equipe de 5 integrantes. A proposta é oferecer uma plataforma simples, acessível e interativa para estudantes, reunindo conteúdos educacionais de diversas áreas do conhecimento.

O sistema foi construído utilizando tecnologias de **desenvolvimento web front-end**, com foco em organização de conteúdo, usabilidade e interatividade.

---

## 🎯 Objetivo

Criar um ambiente digital onde usuários possam:

* Acessar materiais de estudo organizados por disciplinas
* Testar seus conhecimentos através de quizzes interativos
* Navegar de forma intuitiva entre diferentes áreas do saber

---

## 📖 Disciplinas Disponíveis

O portal conta com conteúdos nas seguintes áreas:

* 🎨 Artes
* 🔬 Ciências
* ➗ Matemática
* 🎭 Filosofia
* 🌎 Geografia
* 📗 História
* 🗣 Inglês
* 📙 Português
* 📚 (Outras disciplinas podem ser adicionadas futuramente)

---

## 📝 Funcionalidades

* 📄 Páginas dedicadas para cada matéria
* 🧪 Quizzes interativos para fixação de conteúdo

## 🚀 Como Executar o Projeto

1. Clone este repositório:

   ```bash
   git clone https://github.com/seu-repositorio/bitacademy.git
   ```

2. Acesse a pasta do projeto:

   ```bash
   cd bitacademy
   ```

3. Abra o arquivo `Bitacademy.html` em seu navegador.
